# 学校综合业务服务平台

一个面向义务教育阶段初中学校的综合业务服务平台，采用Node.js技术栈，部署在Linux服务器上，支持1000-3000人规模的中型学校。

## 项目概述

本平台是一个模块化的学校综合业务管理系统，涵盖办公管理、教学管理、德育管理、校园安全、后勤服务、人员管理、校园卫生、家校沟通等8大核心模块。

## 技术架构

### 前端技术栈
- Vue 3 + TypeScript + Vite
- Element Plus (企业级UI组件)
- Pinia (状态管理)
- Vue Router 4 (路由管理)
- Axios (HTTP客户端)
- ECharts (图表可视化)

### 后端技术栈
- Node.js 18+
- Nest.js (TypeScript)
- TypeORM (ORM)
- PostgreSQL (关系型数据库)
- Redis (缓存)
- JWT + Passport.js (认证)
- Swagger/OpenAPI (API文档)

## 核心功能模块

### 1. 办公管理模块
- 公文处理：收文登记、拟稿、会签、审批、归档、流程追踪
- 通知发布：分类通知、定向发布、阅读状态跟踪、推送提醒
- 用章申请：在线申请、审批流程、使用记录、统计分析
- 档案管理：文档分类、版本控制、权限管理、全文检索
- 会议管理：会议室预约、会议通知、会议记录、纪要归档

### 2. 教学管理模块
- 课程安排：课程表管理、冲突检测、调课申请、排课优化
- 作业发布：作业布置、提交、批改、成绩统计、错题本
- 班级管理：班级信息、学生管理、班委管理、班级文化
- 成绩管理：成绩录入、统计分析、成绩单生成、历史对比
- 教师工作：教案管理、教学日志、听课评课、绩效考核

### 3. 德育管理模块
- 学生评价：德育量化评价、综合素质评价、成长档案
- 班主任工作：班级计划、班会记录、学生谈心、家访记录
- 德育活动：活动发布、参与记录、成果展示、学分认定
- 文明班级：评比指标、月度评选、表彰激励、公示展示
- 奖惩管理：表彰奖励、违纪处理、档案记录、统计分析

### 4. 校园安全模块
- 安全排查：隐患上报、整改跟踪、分级预警、统计分析
- 视频监控：摄像头管理、实时查看、录像回放、AI告警
- 消防安全：设备巡检、隐患记录、消防演练、培训记录
- 食品安全：食材溯源、采购记录、检测报告、留样管理
- 应急预案：预案管理、演练记录、应急响应、事件追溯

### 5. 后勤服务模块
- 维修申请：在线报修、工单派发、进度跟踪、满意度评价
- 物品申领：物品目录、申领审批、库存管理、领用记录
- 采购管理：需求汇总、供应商管理、招标流程、合同管理
- 食堂管理：
  - 食材采购：供应商管理、采购订单、验收记录、价格监测
  - 膳食营养：菜谱管理、营养分析、膳食建议、满意度调查
  - 食品安全：检测管理、留样记录、员工健康、明厨亮灶
  - 用餐管理：餐卡管理、预订就餐、营养分析、消费统计
- 资产管理：资产登记、借用归还、盘点管理、折旧计算

### 6. 人员管理模块
- 学生管理：学籍管理、信息维护、学籍变动、毕业档案
- 教师管理：人事档案、工作经历、培训记录、绩效考核
- 职工管理：职工档案、岗位职责、考勤管理、薪资管理
- 组织管理：部门设置、岗位配置、人员编制、组织架构

### 7. 校园卫生模块
- 学生体检：体检安排、数据录入、健康档案、异常跟踪
- 晨午检：每日检查、异常上报、统计分析、传染病防控
- 病因追踪：病假记录、康复跟踪、复课证明、统计分析
- 健康教育：卫生讲座、健康宣传、疾病预防、心理咨询
- 环境监测：教室环境、饮用水质、空气质量、消毒记录

### 8. 家校沟通模块
- 调查问卷：问卷设计、定向发布、结果统计、数据可视化
- 意见建议：意见箱、处理流程、反馈跟踪、满意度评价
- 班级沟通：班级公告、作业通知、成绩推送、活动邀请
- 家长会：邀请发布、在线报名、会议记录、资料共享

## 快速开始

### 环境要求
- Node.js 18+
- PostgreSQL 14+
- Redis 6+
- Docker (可选，用于容器化部署)

### 安装依赖

#### 前端
```bash
cd frontend
npm install
```

#### 后端
```bash
cd backend
npm install
```

### 配置环境变量

#### 前端配置
在 `frontend/.env.development` 中配置开发环境变量：
```env
VITE_API_BASE_URL=http://localhost:3000/api
```

#### 后端配置
在 `backend/.env` 中配置后端环境变量：
```env
# 数据库配置
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=postgres
DB_PASSWORD=your_password
DB_DATABASE=school_management

# Redis配置
REDIS_HOST=localhost
REDIS_PORT=6379

# JWT配置
JWT_SECRET=your_jwt_secret_key
JWT_EXPIRATION=7d
```

### 启动服务

#### 启动前端
```bash
cd frontend
npm run dev
```

#### 启动后端
```bash
cd backend
npm run start:dev
```

### 使用Docker启动
```bash
docker-compose up -d
```

## 开发规范

### 代码规范
- 使用 ESLint + Prettier 进行代码格式化
- 遵循 TypeScript 严格模式
- 提交代码前进行静态检查

### Git 工作流
- 主分支：`main`
- 开发分支：`develop`
- 功能分支：`feature/功能名称`
- 修复分支：`fix/问题描述`

### 提交信息规范
```
feat: 新功能
fix: 修复bug
docs: 文档更新
style: 代码格式调整
refactor: 代码重构
test: 测试相关
chore: 构建/工具链相关
```
